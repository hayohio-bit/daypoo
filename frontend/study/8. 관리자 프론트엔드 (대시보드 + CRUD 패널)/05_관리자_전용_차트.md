# 섹션 5: 관리자 전용 Recharts 고급 차트 설계

**[정합성 검증 완료 사항]**
> `package.json` 통과 및 실제 `AdminPage.tsx` 코드 내부에서 `recharts` 모듈을 활발히 사용하고 있는 것을 검증했습니다. 단순히 라인만 긋는 데 그치지 않고 `<defs>`를 통한 `<linearGradient>`, `<filter>` 드롭 섀도우를 입힌 프리미엄 대시보드 차트가 매우 뛰어나게 완성되어 있습니다.

## 1. 차트 공통 환경 (ResponsiveContainer & Custom Tooltip)

Recharts는 본래 브라우저 크기에 대응하지 못하는 고정 SVG 형태이나, DayPoo 관리자 화면은 `ResponsiveContainer`로 감싼 후 `CustomTooltip`을 적용하여 화려한 블러 효과를 부여했습니다.

```tsx
// CustomTooltip 구현체 (AdminPage.tsx 내부)
const CustomTooltip = ({ active, payload, label }: any) => {
  if (active && payload && payload.length) {
    return (
      <motion.div
        initial={{ opacity: 0, y: 15 }}
        animate={{ opacity: 1, y: 0 }}
        className="p-5 rounded-[28px] border border-white/60 bg-white/30 backdrop-blur-3xl shadow-lg"
      >
        <p className="text-[11px] font-black uppercase text-black/50">{label} Stats</p>
        <div className="space-y-4">
          {payload.map((entry: any, index: number) => (
            <div key={index} className="flex justify-between items-center gap-12">
               <span style={{ color: entry.color }}>{entry.name}</span>
               <span className="font-black">{entry.value.toLocaleString()}</span>
            </div>
          ))}
        </div>
      </motion.div>
    );
  }
  return null;
};
```

## 2. ComposedChart (Area + Bar 멀티 축 복합)

`성장 엔진 리포트 (Growth Velocity)`를 구현하는 핵심 요소입니다. 신규 가입자 트래픽(Area) 위에 유료 매출 발생량(Bar)을 겹쳐서 비즈니스의 인과관계를 하나의 그래프에 투영했습니다.

```tsx
// ComposedChart + 그라데이션 SVG Filter 커스터마이징
<ResponsiveContainer width="100%" height="100%">
  <ComposedChart data={trendData}>
    <defs>
      <linearGradient id="mainGradient" x1="0" y1="0" x2="0" y2="1">
        <stop offset="0%" stopColor="#1B4332" stopOpacity={0.15} />
        <stop offset="100%" stopColor="#1B4332" stopOpacity={0} />
      </linearGradient>
      {/* 바 차크 황금색 그라데이션 */}
      <linearGradient id="barGradient" x1="0" y1="0" x2="0" y2="1">
        <stop offset="0%" stopColor="#E8A838" stopOpacity={0.8} />
        <stop offset="100%" stopColor="#E8A838" stopOpacity={0.1} />
      </linearGradient>
    </defs>
    
    <CartesianGrid strokeDasharray="12 12" vertical={false} />
    <XAxis dataKey="name" axisLine={false} tickLine={false} />
    <Tooltip content={<CustomTooltip />} />
    
    <Bar
      dataKey="sales"
      name="유료 결제"
      fill="url(#barGradient)"
      radius={[6, 6, 0, 0]}
      barSize={24}
    />
    <Area
      type="monotone"
      dataKey="users"
      name="신규 방문"
      stroke="#1B4332"
      fill="url(#mainGradient)"
      activeDot={{ r: 8, strokeWidth: 0, fill: "#000" }}
    />
  </ComposedChart>
</ResponsiveContainer>
```

> [!WARNING] 차트 렌더링 주의점
> 차트에 삽입된 `filter="url(#neonGlow)"` 같은 SVG 필터는 화려하긴 하나 GPU 리소스를 많이 소모합니다. 30일치 이상의 대량의 데이터를 매핑할 경우 모바일 화면에서 버벅임(Frame Drop) 현상이 보고되어, 현재 코드에서는 `chartRange === '30D' ? 'none' : 'url(#barShadow)'` 조건절을 걸어 필터를 해제하는 최적화가 적용되어 있습니다. 훌륭한 접근입니다.
