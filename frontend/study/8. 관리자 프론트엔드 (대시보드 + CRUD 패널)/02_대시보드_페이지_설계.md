# 섹션 2: 대시보드 (Dashboard) 페이지 상세 설계

**[정합성 검증 완료 사항]**
> `AdminPage.tsx` 내 탑재된 `DashboardView` 컴포넌트를 확인했습니다. `Bento Grid` 방식과 Framer Motion 도입, Recharts 등을 모두 활용한 프리미엄 디자인 설계가 매우 인상적으로 구현되어 있습니다.

## 1. Bento Grid 레이아웃 & StatsGrid (KPI 지표)

상단에 4개의 주요 KPI 지표들을 보여주며 `GlassCard`라는 통일된 UI 컴포넌트로 감싸 시각적 만족도를 높인 상태입니다.

```tsx
// DashboardView 탑재 기능 일부
<div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-4 gap-6">
  <StatWidget
    title="현재 접속자"
    value={liveUsers}
    trend="+12%"
    isUp={true}
    color={COLORS.info}
    icon={Activity}
    progress={78} // 게이지 바
    badge="Live"
  />
  <StatWidget
    title="누적 사용자"
    value={stats?.totalUsers.toLocaleString()}
    trend="+2.4%"
    isUp={true}
    color={COLORS.primary}
    icon={Users}
  />
  <StatWidget
    title="미답변 문의"
    value={stats?.pendingInquiries}
    isUp={false} // 문의수가 느는건 나쁜거이므로
    color={COLORS.error}
    icon={MessageSquare}
    badge="Urgent"
  />
</div>
```

## 2. StatWidget 내부 UI/UX 요소

숫자만 보여주는 것이 아니라, Framer Motion 에어레이션과 하단 `Efficency Index` 바(게이지)를 부착해 데이터가 살아 숨쉬는 느낌을 주고 있습니다.

```tsx
const StatWidget = ({ title, value, trend, isUp, icon: Icon, color, progress = 0, badge }: any) => {
  return (
    <GlassCard glowColor={`${color}15`} className="group hover:-translate-y-1.5 transition-all">
      <div className="flex justify-between items-start mb-6">
        <div style={{ background: `${color}10`, color }}>
          <Icon size={24} />
        </div>
        <div className="flex flex-col items-end gap-2">
           {badge && <span className="bg-black text-white px-2 py-0.5 rounded-full">{badge}</span>}
           <div className={`flex items-center gap-1 ${isUp ? 'text-emerald-600' : 'text-rose-600'}`}>
              <TrendingUp size={12} className={!isUp ? "rotate-180" : ""} />
              {trend}
           </div>
        </div>
      </div>
      
      {/* 수치 */}
      <span className="text-4xl font-black text-black">{value}</span>
      
      {/* 게이지 바 애니메이션 */}
      <div className="h-2 w-full bg-black/5 rounded-full overflow-hidden mt-4">
        <motion.div
           initial={{ width: 0 }}
           animate={{ width: `${progress}%` }}
           transition={{ duration: 1.5 }}
           className="h-full rounded-full"
           style={{ backgroundColor: color }}
        />
      </div>
    </GlassCard>
  );
}
```
