# 📈 관리자 대시보드 차트 시각적 피로도 개선 및 성능 최적화 계획

30D(30일) 데이터의 과도한 밀도로 인한 시각적 불쾌감(징그러움)과 렌더링 지연(렉) 문제를 해결하기 위해 디자인과 데이터를 최적화합니다.

## 🛠️ 수정 사항

### 1. 데이터 밀도 최적화 (Sampling)

- **30D 탭 데이터 축소**: 30일치 모든 데이터를 개별 막대로 노출하는 대신, **격일(Every other day)** 또는 **주요 지점**으로 샘플링하여 막대 사이의 간격을 확보합니다.
- **X축 레이블 간소화**: 날짜를 빽빽하게 적지 않고 균형 있게 배치하여 '휑하지 않으면서도 시원한' 느낌을 줍니다.

### 2. 시각적 간결화 (30D 전용 스타일)

- **30D 모드**: 막대(Bar)의 투명도를 높이거나 굵기를 가늘게 하여 '세밀함'보다는 '전체적인 흐름(Area Chart)'에 집중하도록 유도합니다.
- **7D 모드**: 기존의 풍부한 애니메이션과 스타일을 유지하여 '상세 리포트' 느낌을 살립니다.

### 3. 성능(FPS) 및 애니메이션 개선

- **애니메이션 가중치 조절**: 전체 `animationDuration`을 1.5s → **0.6s**로 단축하여 즉각적인 반응성을 제공합니다.
- **무거운 필터 동적 비활성화**: 30D 모드와 같이 데이터가 많은 경우 복잡한 SVG 필터(Shadow/Glow)를 조건부로 비활성화하여 렉을 제거합니다.
